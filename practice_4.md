# 1 MVCの役割を構成する、Railsの各コンポーネントの名前を列挙し、それぞれの役割を説明したください。

- Active Record  
モデルの機能の補助
- Action Controller  
コントローラーの補助
- Action view  
ビューに関する機能の補助
- Action dispauch  
ルーティングに関する機能の補助

## 正解

- Active Record  
役割：リソース管理
- Action View  
役割：ユーザーインターフェース
- Action Controller  
役割：リソース制御
- Action Dispatch(ルーター)  
役割：Webリクエストの解析とルーティング
