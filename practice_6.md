返り値が真偽値なので、scaffoldでは条件分岐に使い、文章中でcountとfull_messagesメソッドを使用して表示している。
```ruby
if オブジェクト.errors.eny?
  オブジェクト.errors.count
  オブジェクト.errors.full_messages.each do | message |
    message
  end
end
```
